# Schemas

This directory contains JSON schemas for evidence and artifact contract validation.

Return to the project landing page: [`../../README.md`](../../README.md). See [`../evidence-schema-contracts.md`](../evidence-schema-contracts.md) for the narrative schema contract.

## Files

- `artifact-manifest.schema.json`, schema for artifact manifest records
- `evidence-record.schema.json`, schema for evidence records

Generated lab and validation packages commonly include `artifact-manifest.json`, `SHA256SUMS.txt`, JSON or JSONL evidence, reports, archives, and `.sha256` sidecars.
