# Reports

This directory is the default location for generated local reports.

Return to the project landing page: [`../README.md`](../README.md). See evidence expectations in [`../docs/evidence-schema-contracts.md`](../docs/evidence-schema-contracts.md).

Generated report contents are ignored by Git, except for the directory placeholder and this index. Typical outputs may include Markdown reports, JSON or JSONL evidence, `artifact-manifest.json`, `SHA256SUMS.txt`, archives, and `.sha256` sidecars.

Do not commit generated evidence from local runs unless a specific review workflow asks for a sanitized artifact.
